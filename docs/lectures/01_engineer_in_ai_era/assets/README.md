# Материалы для лекции 1

Этот каталог содержит заранее подготовленные артефакты, которые в сценарии лекции ([../script.md](../script.md)) выглядят как результат работы AI «прямо на глазах у аудитории», но фактически подготовлены заранее — чтобы исключить накладки живой генерации во время занятия.

## Файлы

- [`ai_proposal.md`](ai_proposal.md) — предложение AI для муниципального экологического центра: промпт и полный ответ (архитектура, схема данных, фрагмент кода, описание панели). Используется в **блоке 1** сценария («Встреча с ситуацией»).
- [`misleading_dashboard_example.md`](misleading_dashboard_example.md) — пример «убедительной панели неверных данных»: два несопоставимых источника, наложенные на один график. Используется в **блоке 3** сценария («Разрыв артефакт ↔ результат»).

## Как использовать на занятии

Оба артефакта показываются студентам как готовый результат — без объяснения, что они подготовлены заранее. Разбор того, что в них скрыто и почему это проблема, происходит только после того, как аудитория успела отреагировать на артефакт как есть (см. тайминг в сценарии).

## Статус

Черновик. Технические детали (стек, конкретные значения) выбраны ради правдоподобия примера, а не как рекомендация архитектуры для реального пилота. Не предназначены для использования как основа кода `pe2030-air-quality-pilot`.
