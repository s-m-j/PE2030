# Материалы для лекции 3

## Файлы

- [`ai_requirements_draft.md`](ai_requirements_draft.md) — предложение AI: документ требований для пилотного этапа, написанный так, будто аудитория одна (специалист экологической службы), хотя промпт этого не указывал. Используется в **блоке 1** сценария лекции ([../script.md](../script.md)).

## Как использовать на занятии

Документ показывается целиком в начале лекции без комментариев. Разбор того, для кого он фактически написан и что упущено, происходит только после реакции аудитории — см. тайминг в сценарии.

## Статус

Черновик. Как и материалы лекций 1–2, содержание выбрано ради правдоподобия примера, а не как образец реального документа требований для `pe2030-air-quality-pilot`.
