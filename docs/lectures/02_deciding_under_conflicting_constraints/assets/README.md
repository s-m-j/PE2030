# Материалы для лекции 2

Этот каталог содержит заранее подготовленные артефакты для лекции 2 и последующего семинара с ролевой игрой «заказчик — исполнитель».

## Файлы

- [`stakeholder_cards.md`](stakeholder_cards.md) — закрытые карточки шести сторон конфликта: видимые всем требования (используются в лекции, блок 1 [`../script.md`](../script.md)) и скрытые детали, известные только исполнителю каждой роли (используются на семинаре).
- [`ai_comparison_matrix.md`](ai_comparison_matrix.md) — предложение AI: сравнительная матрица трёх альтернатив с весовыми коэффициентами и однозначной рекомендацией. Используется в **блоке 6** сценария лекции.

## Как использовать на занятии

`ai_comparison_matrix.md` показывается как есть — без комментария, что веса подобраны произвольно. Разбор того, что в нём скрыто, происходит только после реакции аудитории (см. тайминг в сценарии).

`stakeholder_cards.md` на лекции используется только частично (видимые требования, слайд 2 в `../slides.md`). Скрытые детали карточек не должны попадать в руки студентов до семинара — иначе теряется смысл ролевой игры.

## Статус

Черновик. Как и в лекции 1, технические и организационные детали выбраны ради правдоподобия примера, а не как предписание для реального пилота `pe2030-air-quality-pilot`.
