# Материалы для лекции 5

## Файлы

- [`ai_generated_giant_pr.md`](ai_generated_giant_pr.md) — предложение AI: единый Pull Request, реализующий ADR из лекции 4, смешанный с посторонним рефакторингом, незапрошенным кешированием и переименованиями. Используется в **блоках 1–2** сценария ([../script.md](../script.md)) — раздаётся аудитории с ограничением по времени на «ревью».
- [`decomposed_pr_example.md`](decomposed_pr_example.md) — контрпример: то же решение, реализованное за четыре независимых, проверяемых шага, каждый со ссылкой на ADR. Используется в **блоке 7**.

## Как использовать на занятии

`ai_generated_giant_pr.md` раздаётся или показывается целиком в начале лекции. Ключевая часть упражнения — реально ограничить время на «ревью» (3 минуты, см. сценарий), чтобы аудитория на себе почувствовала цену непроверяемого объёма, а не просто услышала тезис об этом.

## Статус

Черновик, учебный пример. Код и PR написаны ради иллюстрации разницы между единым и декомпозированным изменением, а не как образец реального процесса разработки в `pe2030-air-quality-pilot`.
